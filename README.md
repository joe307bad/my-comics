## ComicTracker

Keep track of all the comics you read, gain points, earn badges, recommend issues/volumes to friends, and follow latest releases.
 
 Built with:
 
 - Ionic 3/Angular 4
 - ngrx 4
 - Immutable.js
 - Karma/Jasmine/Protractor
 - Moment.js
 - (Coming soon) PouchDB/CouchDB
 - (Coming soon) TravisCI
